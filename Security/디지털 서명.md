# 디지털 서명

디지털 서명은 비대칭키 암호화 기법을 사용하여 송신자의 신원을 증명하고, 문서의 변조를 막는 역할을 한다.

### 구성 알고리즘

디지털 서명은 아래 3가지의 알고리즘으로 구현된다.

1. 공개키 생성 알고리즘
2. 송신자 비밀키로 서명을 생성하는 알고리즘
3. 공개키로 서명을 검증하는 알고리즘

### 서명 검증 방법

- 송신자
  1. 원본 데이터를 해싱한다.
  2. 해시 값을 비밀키로 암호화하여 서명을 생성한다.
  3. 원본 데이터와 서명을 전송한다.
- 수신자
  1. 받은 데이터를 해싱한다.
  2. 서명을 공개키로 복호화한다.
  3. 데이터를 해싱한 값과 서명을 복호화해 얻은 해싱 값이 동일한지 확인한다.
